# Mesto Russia — Production Build

**⚠️ ВАЖНО: В этом репозитории размещена только production-версия приложения.**  
Код для разработки хранится в отдельном приватном репозитории.

## 📝 Описание

Mesto — веб-приложение для публикации и просмотра фотографий. Пользователь может изменять данные профиля, создавать и удалять карточки, а также ставить отметки «нравится». Все ключевые действия выполняются через API, где сохраняются сведения о пользователях и карточках.

## ✨ Функциональность
- 📱 Корректное отображение на разных размерах экрана
- ✏️ Обновление профиля (имя, род занятий, аватар)
- ➕ Создание новых карточек с фотографиями
- ❌ Удаление карточек, созданных пользователем
- ❤️ Добавление и снятие лайков
- 🔍 Открытие изображений в полноэкранном просмотре
- ✅ Проверка полей форм перед отправкой
- 🔄 Интеграция с сервером по REST API
- ⏳ Понятная индикация загрузки в формах
- 🗑️ Подтверждение перед удалением карточки
- 🔢 Вывод актуального количества лайков

## 🛠 Технологии
- **HTML5** (семантическая структура страниц)
- **CSS3** (Flexbox, Grid, адаптив, методология БЭМ)
- **JavaScript** (ES6+, ООП-подход, асинхронная работа с сетью)
- **Vite** (сборка и оптимизация проекта)
- **REST API** (обмен данными с серверной частью)
- **Git** (ведение истории изменений)
- **GitHub Actions** (автоматизированный деплой)
